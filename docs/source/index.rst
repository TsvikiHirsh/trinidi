TRINIDI Documentation
=====================

`Time-of-Flight Resonance Imaging with Neutrons for Isotopic Density Inference (TRINIDI) <https://github.com/lanl/trinidi/tree/develop>`__ is a Python package for estimating isotopic densities using neutron transmission resonance images.

This code is based on the work in `this manuscript <https://arxiv.org/abs/2302.12577>`__ and it uses SCICO :cite:`balke-2022-scico`.


TRINIDI provides cross section data that is extracted from the `Lib80x <https://nucleardata.lanl.gov/ace/lib80x/>`__ library :cite:`conlin2018release`.




*Note that TRINIDI is currently under construction.*
